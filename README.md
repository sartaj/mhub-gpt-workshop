# Requirements

- python+pip
- node+npm

# Setup

Run `sh setup.sh`.

## To run web app

from `web-app`, `OPENAI_API_KEY="yourapikey" npm run dev`

## To run GPT script

from `gpt-integration`, `OPENAI_API_KEY="yourapikey" python run-gpt.py "Give me the top 5 patients with a high risk of heart disease."`
